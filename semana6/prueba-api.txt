La ruta POST /registro recibe información enviada por el usuario en formato JSON. Cuando se envían los campos "nombre" y "mensaje", el servidor procesa los datos mediante req.body y responde con un objeto JSON que confirma la recepción de la información.

Por ejemplo, al enviar:

{
"nombre": "Maria",
"mensaje": "Hola comunidad"
}

el servidor responde:

{
"estado": "Datos recibidos",
"nombre": "Maria",
"mensaje": "Hola comunidad"
}
