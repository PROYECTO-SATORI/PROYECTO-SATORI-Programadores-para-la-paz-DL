Sistema operativo:
Es el software base que permite ejecutar Node.js y gestionar los recursos del hardware como memoria y procesador.

Node.js:
Es el entorno que permite ejecutar JavaScript en el servidor para crear aplicaciones backend.

Aplicación server.js:
Es el archivo principal donde se define la lógica del servidor, rutas y respuestas.

Puerto:
Es el punto de entrada donde el servidor escucha solicitudes. En este caso se usa el puerto 3000.

Logs:
Son registros que muestran eventos del sistema, como inicios, errores o actividad del servidor.
